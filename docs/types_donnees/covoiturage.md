## Principaux types de données

### Utilisation des plateformes de covoiturage
    *   Description : Nombre de trajets, nombre d'utilisateurs, distances parcourues, lieux de départ et d'arrivée, taux d'occupation des véhicules.
    https://observatoire.covoiturage.gouv.fr/ 
    Formats : Données agrégées, API (selon les plateformes), rapports.
    *   principaux opérateurs :
        *   **BlaBlaCar :** [https://www.blablacar.fr/](https://www.blablacar.fr/) (Données agrégées et/ou sur demande, pas toujours publiques)
        *   **Klaxit :** [https://www.klaxit.com/](https://www.klaxit.com/) (Données potentiellement disponibles via partenariats avec les collectivités)
        *   **Mobicoop :** [https://www.mobicoop.fr/](https://www.mobicoop.fr/) (Coopérative de covoiturage, données potentiellement plus ouvertes)

### Enquêtes et études sur le covoiturage
    *   Description : Profil des covoitureurs, motivations, impacts environnementaux, etc.
    *   Sources :
        *   **ADEME :** [https://www.ademe.fr/](https://www.ademe.fr/) (Études et publications sur le covoiturage)
        *   **Cerema :** [https://www.cerema.fr/](https://www.cerema.fr/) (Études et données sur la mobilité)
        *   **Collectivités territoriales :** (Enquêtes locales, études d'impact)
    *   Formats : Rapports, publications, données d'enquêtes (parfois disponibles).

### Aires de covoiturage :**
    *   Description : Localisation, capacité, équipements des aires de covoiturage.
    *   Sources :
        *   **OpenStreetMap :** [https://www.openstreetmap.org/](https://www.openstreetmap.org/) (Données collaboratives, potentiellement incomplètes)
        *   **Collectivités territoriales :** (Inventaires des aires de covoiturage)
    *   Formats : GeoJSON, Shapefile, KML.
