## Types de données (à revoir et compléter!)

### Utilisation des plateformes

*   Description : Nombre de locations, nombre d'utilisateurs, durée moyenne d'utilisation, lieux de prise et de retour des véhicules, etc.
*   Sources :
    *   **Free2Move :** [https://www.free2move.com/](https://www.free2move.com/) (Données potentiellement disponibles via partenariats)
    *   **Getaround :** [https://www.getaround.com/](https://www.getaround.com/) (Ancien Drivy, données potentiellement disponibles via partenariats)
    *   **Communauto :** [https://www.communauto.com/](https://www.communauto.com/) (Données potentiellement disponibles, surtout au Québec et dans certaines villes en France)
*   Formats : Données agrégées, API (selon les plateformes), rapports.

### Localisation des stations et véhicules

*   Description : Emplacement des stations d'autopartage, disponibilité des véhicules en temps réel.
*   Sources :
    *   **OpenStreetMap :** [https://www.openstreetmap.org/](https://www.openstreetmap.org/) (Données collaboratives, potentiellement incomplètes)
    *   **Applications des opérateurs :** (Free2Move, Getaround, Communauto)
*   Formats : GeoJSON, API, KML.

### Enquêtes et études

*   Description : Profil des utilisateurs, impacts sur la mobilité, aspects économiques, etc.
*   Sources :
    *   **ADEME :** [https://www.ademe.fr/](https://www.ademe.fr/) (Études et publications sur les nouvelles mobilités)
    *   **Cerema :** [https://www.cerema.fr/](https://www.cerema.fr/) (Études et données sur la mobilité)
    *   **Collectivités territoriales :** (Enquêtes locales, études d'impact)
*   Formats : Rapports, publications, données d'enquêtes (parfois disponibles).

### Tarification et modèles économiques

*   Description : Structure des tarifs, coûts d'utilisation, rentabilité des services.
*   Sources :
    *   **Sites web des opérateurs :** (Free2Move, Getaround, Communauto)
    *   **Études de marché :** (Réalisées par des cabinets spécialisés)
*   Formats : Tableaux de tarifs, rapports.
