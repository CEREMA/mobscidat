Ce code contient le site web de partage de ressources sur les données de mobilité
Initié par le Cerema dans le cadre du projet MobSciDat Factory (contrat n°ANR-23-PEMO-0004)
visible à l'adresse [https://cerema.github.io/mobscidat/](https://cerema.github.io/mobscidat/)

Ce travail bénéficie d'une aide de l'Etat gérée par l'Agence Nationale de la Recherche attribuée au project Mob Sci-Dat Factory au titre de France 2030 portant la référence ANR-23-PEMO-0004.
